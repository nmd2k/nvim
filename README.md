# My neo-vim config


## Installation

**For Windows**
* Release version:
```
winget install Neovim.Neovim
```

**For MacOS**
Install release version using Homebrew:

```
brew install neovim
```

**For Ubuntu**

```
sudo apt install neovim
```

## Adding config

Git clone this repo into `~/.config/nvim`

```bash
git clone https://github.com/nmd2k/nvim.git ~/.config/nvim
```
